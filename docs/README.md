# Documentation

This folder contains user-facing and maintainer-facing PCAT documentation.

## GitHub Pages

This folder is ready to be used as the GitHub Pages source.

Recommended repository setting:

- Source: deploy from a branch.
- Branch: `main`.
- Folder: `/docs`.

The public landing page is `index.html`. Static assets live in `assets/`.

## Reference

- `reference/PCAT_ARCHITECTURE.md`: product philosophy, architecture, design decisions, contribution model, and implemented vs planned scope.
- `reference/PCAT_TECHNICAL_REFERENCE.md`: complete technical reference for commands, data models, outputs, findings, artifacts, and planned features.
- `reference/PCAT_MANUAL.md`: command manual and systematic test reference.
- `reference/PCAT_FUTURE_CTF_UPDATE.md`: current plans for future CTF features.
- `reference/PCAT_ROADMAP_AND_DECISIONS.md`: plans and progress roadmaps of the tool, with explanations of decisions.
